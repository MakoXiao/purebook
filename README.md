

# 清阅
一款网络小说阅读器📕,采用纯flutter编写，支持追书、看书，仿真翻页等功能
## 使用技术 
dart,provider,eventbus,dio等

## TIP

* 服务器是阿里云1Core2G弱鸡服务器,请各位大神手下留情不要攻击或者爬取数据,供大家学习使用。。
